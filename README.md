# Number Guessing Game

This is a simple Number Guessing Game built using JavaScript and Node.js.

I created this project while learning JavaScript to practice important concepts such as:

* Variables
* Loops
* Conditional Statements (if-else)
* User Input Handling
* Random Number Generation
* Basic Game Logic

The game generates a random number between 1 and 100, and the user has to guess the correct number. After each guess, the game provides hints whether the guessed number is higher or lower than the random number. It also tracks the number of attempts and calculates a score.

## Technologies Used

* JavaScript
* Node.js
* prompt-sync

## How to Run

1. Install dependencies:
   npm install prompt-sync

2. Run the game:
   node game.js

This project was created for learning and practicing JavaScript fundamentals.
